## 🚀 How to Deploy (Quick Steps)

### Step 1: Create Railway Account (2 min)
```
Go to: https://railway.app
Sign up with GitHub
```

### Step 2: Deploy Repository (5 min)
```
1. Click "New Project"
2. Select "Deploy from GitHub repo"
3. Choose your AquaSmart repository
4. Click "Deploy Now"
```

### Step 3: Configure Variables (2 min)
```
Backend:
- GROQ_API_KEY=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
- USE_AWS=false
- FALLBACK_ENABLED=true
- APP_ENV=production

Frontend:
- BACKEND_URL=https://your-backend-url
```

### Step 4: Get URLs & Test (3 min)
```
- Frontend URL: Your hackathon submission link
- Backend URL: For API access
- Test: Visit frontend URL in browser
```

```
Copy frontend URL and submit!
```
